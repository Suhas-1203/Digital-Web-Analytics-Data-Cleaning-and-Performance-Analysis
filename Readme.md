# Web Analytics Performance Dashboard

## Overview
This project presents a Power BI dashboard that analyzes website performance across different traffic channels. It helps identify high-performing channels based on revenue, user engagement, and conversion efficiency.

## Tools Used
- Power BI
- Python (for data cleaning)
- Excel / CSV

## Key Metrics
- Total Users
- Total Sessions
- Total Revenue
- Conversion Rate
- Revenue per Session

## Dashboard Features
- Revenue share by channel
- Top channels by revenue
- Monthly revenue trend by channel
- Users by channel
- Conversion rate by channel
- Interactive slicer for channel filtering

## Files
- `dashboard/` → Power BI (.pbix) file  
- `data/` → Cleaned dataset  
- `screenshots/` → Dashboard images  

## How to View
1. Download the `.pbix` file
2. Open using **Power BI Desktop**
